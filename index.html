Quick solutions to common issues

Module doesn't appear in list

  • Verify folder location: modules/addons/pynoxi_mpwa/
  • Check permissions (644 files, 755 directories)
  • Ensure all files are present

Messages not being sent

  • Verify API URL and API Key correctness
  • Check sender number formatting (no spaces/symbols)
  • Ensure templates are activated
  • Review Logs tab for error details

API connection errors

  • Verify cURL extension is enabled
  • Check HTTPS outbound connections
  • Confirm firewall isn't blocking requests
